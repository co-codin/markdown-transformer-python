## [Unreleased]

### Added
- Initial release of any-to-md document converter
- Support for 15+ document formats via Pandoc and Docling
- Asynchronous task processing with status tracking
- Image extraction and organization
- Optional S3 integration for cloud storage
- Docker support with optimized builds
- Python client library
- Comprehensive API documentation
- Health monitoring endpoint
- Auto-cleanup of old conversion results

### Supported Formats
- **Pandoc**: ODT, EPUB, HTML, HTM, RTF, DOC, DOCX, XLS
- **Docling**: PDF, PPTX, XLSX

### Security
- Input validation for file uploads
- Path traversal protection
- Configurable file size limits
- Secure error handling

### Documentation
- Comprehensive README with quick start guide
- API documentation via Swagger/ReDoc
- Contributing guidelines
